# 📊 Stack Overflow Trends Analysis (2008–2024)

## 📌 Overview

This project analyzes the trend of Stack Overflow questions from 2008 to 2024. It visualizes monthly question counts along with a 12-month rolling average and overlays key AI milestones.

## 🚀 Features

- Monthly question trends
- 12-month rolling average smoothing
- AI milestone annotations (Transformer, GPT-3, ChatGPT, GPT-4)

## 📂 Data Source

Data was obtained using Stack Exchange Data Explorer via SQL query.

## 🧠 Key Insight

The analysis shows a decline in Stack Overflow activity in recent years, potentially correlated with the rise of AI tools like ChatGPT.

## 🛠️ Tech Stack

- Python
- Pandas
- Matplotlib

## ▶️ How to Run

```bash
pip install -r requirements.txt
python plot_stackoverflow.py
```
